# UVM_Testbenches
- Collection of UVM testbenches starting with basic combinational and sequential blocks.
- Building core verification skills through structured environments, coverage, and assertions.
- Working towards expanding toward more complex designs such as controllers, FIFOs, and bus interfaces.
- Focused on developing scalable, industry-aligned UVM methodology.

Testbenches Written
1. 4-to-1 Mux
2. 3-to-8 Decoder
3. Comparator
4. Adder
5. ALU
6. Barrel Shifter
7. Sequential Up-Down Counter
8. Universal Shift Register
9. Traffic Light Controller 
10. Synchronous FIFO
11. APB RAM

Working towards - 
1. Parameterizing these components
2. Including Assertions 

I'm open and grateful to feedback on these testbenches, and areas to improve!
I would also love suggestions on what designs to write testbenches for, to improve my skills and knowledge as a Verification Engineer.
